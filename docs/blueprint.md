# **App Name**: ResumePilot AI

## Core Features:

- User Authentication: Secure user registration and login using Firebase Authentication.
- Resume Upload and Storage: Upload resume in PDF format and store it securely in Firebase Storage.
- Job Details Input: Allow users to input the job role and description for targeted analysis.
- AI-Powered Resume Analysis: Analyze the resume against the job description using Google Gemini API to extract skills and identify weaknesses.
- ATS Score Generation: Generate an ATS score (0-100) based on skills relevance, keyword match, experience presence, and education using Gemini as a tool.
- AI Feedback and Suggestions: Provide detailed feedback and suggestions for resume improvement generated by Gemini.
- Display ATS Results: Display ATS score, AI feedback, and missing skills in a clear, easy-to-understand format.
- Data persistence: Persist and display user information and resume analysis using Firestore

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#F0F2F5), a desaturated version of the primary, to provide a clean and modern backdrop.
- Accent color: Blue-Violet (#7986CB) to complement the primary color and highlight important actions and elements.
- Body font: 'Inter', a grotesque sans-serif, to ensure readability and a clean, modern feel. Headline Font: 'Space Grotesk', a sans-serif, used for a techy, scientific feel
- Code font: 'Source Code Pro' for displaying code snippets, used in descriptions of needed skills
- Use modern, professional icons to represent different skills and resume sections.
- Implement a clean, card-based layout for easy navigation and information presentation.
- Subtle animations for loading states and transitions to improve user experience.
